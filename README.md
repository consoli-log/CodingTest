## ✅ 문제 풀이 규칙

- 파일명은 `문제이름.java`
- 클래스 주석에 문제 링크 + 유형 작성
- 커밋 메시지: `[백준/프로그래머스] 문제번호 - 문제이름 (유형)` 형식
